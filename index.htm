Pengetahuan singkat tentang Media NAS

  • Apa itu Media NAS?

    • Media NAS adalah solusi NAS (Network Attached Storage) yang dibuat dengan HTML dan IIS (Internet Information Services)

  • Apa fungsi dari Media NAS?

    • Untuk streaming dan berbagi file di jaringan lokal

  • Apa ekstensi file yang didukung Media NAS?

    • Untuk file bukan media, semua jenis ekstensi didukung.

    Gambar*: .apng, .gif, .ico, .cur,.jpg, .jpeg, .jfif, .pjpeg, .pjp, .png, .svg

    Video*: .mp4, .webm, .ogg

    Audio*: .mp3, .wav, .ogg

    *ekstensi yang didukung untuk streaming
